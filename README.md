# 🏡 Home Rental Website

A modern, responsive **Home Rental Website** built using **HTML5** and **CSS3**. The application provides an elegant user interface for browsing rental properties, exploring featured homes, and learning about available rental services through a clean and intuitive design.

This project was developed to strengthen frontend development skills, with a focus on responsive layouts, semantic HTML, modern CSS techniques, and user-centered interface design.

---

# 📖 Overview

The Home Rental Website is a static frontend web application that simulates a real estate rental platform. It enables users to explore rental listings, discover featured properties, and navigate through a visually appealing interface optimized for desktop, tablet, and mobile devices.

The project emphasizes modern UI/UX principles, responsive design, and organized code structure while providing an engaging browsing experience.
